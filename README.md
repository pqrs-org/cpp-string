# cpp-string

Utilities of `std::string`.

## Requirements

cpp-string depends the following classes.

- [nemtrif/utfcpp](https://github.com/nemtrif/utfcpp).

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
